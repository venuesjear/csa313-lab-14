# Программ хангамжийн чанарын баталгаа ба туршилт хичээлийн лабораторийн ажил №14

Энэ төсөл нь зөвхөн backend нь Java хэл дээр Spark Java REST API ашиглан бичигдсэн жижиг хэмжээний ***Номын сангийн бүртгэлийн систем*** юм.

## Төслийг ажиллуулах заавар

Төслийг ажиллуулахын тулд эхлээд доорх коммандыг ажиллуулж шаардлагатай dependency, plugin-уудыг татаж авна.
```commandline
mvn clean install
```

Нэгжийн тест ажиллуулахдаа доорх коммандыг ажиллуулах бөгөөд ***src/test/java*** хавтаст нэгжийн тестүүд байгаа. Үүнийг ажиллуулснаар Coverage Report мөн автоматаар үүсгэх бөгөөд ***target/site/jacoco/index.html***-ийг browser дээрээ нээж тайланг харна. 

```commandline
mvn test
```

Кодын загвар шалгахдаа доорх коммандыг ашиглах бөгөөд Google Java Style-д нийцсэн эсэхийг шалгана.

```commandline
mvn checkstyle:check
```
